# iceFind ( BTC, ETH, LTC, DOGE, ... )

iceFind is a fast and efficient bitcoin private key brute force tool written in python.

### Quick start

    $ git clone https://github.com/pwdex/iceFind
    $ cd iceFind
    
    $ python3 iceFind.py

Change Script Configurations at: "config.json"
